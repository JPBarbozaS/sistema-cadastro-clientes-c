# Sistema de Cadastro de Clientes em C

Projeto desenvolvido com o objetivo de praticar lógica de programação, estruturas de dados e manipulação de informações utilizando a linguagem C.

## Funcionalidades

* Cadastro de clientes (nome e idade)
* Listagem de clientes cadastrados
* Busca de clientes por nome

## Tecnologias utilizadas

* Linguagem C
* Structs (estrutura de dados)
* Arrays
* Controle de fluxo (switch, loops)

## Aprendizados

Durante o desenvolvimento, foram trabalhados conceitos como:

* Organização de dados com struct
* Manipulação de arrays
* Entrada e tratamento de dados (fgets, scanf)
* Comparação de strings (strcmp)
* Estruturação de um sistema interativo via terminal

## Possíveis melhorias

* Implementar busca sem sensibilidade a maiúsculas/minúsculas
* Adicionar validação de entrada de dados
* Implementar edição e remoção de clientes
* Melhorar a organização do código em funções

## Desafios enfrentados

* Manipulação de strings em C, especialmente no uso de `fgets` e tratamento do caractere de quebra de linha (`\n`)
* Controle do fluxo do programa utilizando `switch` e loops
* Comparação de strings com `strcmp` e entendimento de como funciona a igualdade em C
